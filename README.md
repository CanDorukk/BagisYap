# bagisyap


## Yasal Uyarı
Bu uygulama bağış toplamaz ve herhangi bir ödeme işlemi gerçekleştirmez.  
Yalnızca Türkiye’deki kamu kurumlarının bağış sayfalarına ait, herkese açık olarak paylaşılan bilgiler ve resmî bağlantıları sunar.


## Disclaimer
This application does not collect donations or process payments.
It only provides publicly available information and official links
to donation pages of public institutions in Türkiye.
